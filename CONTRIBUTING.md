# Contribuir a Telegram AI Companion Bot

Gracias por contribuir a este proyecto.

## Pasos para colaborar
1. Haz fork del repositorio.
2. Crea una rama clara:
   ```powershell
git checkout -b feature/nombre
```
3. Realiza cambios pequeños y explícitos.
4. Asegúrate de que el bot siga funcionando.
5. Crea un pull request hacia `main`.

## Formato de commits
- `feat:` nueva funcionalidad
- `fix:` corrección de errores
- `docs:` cambios de documentación
- `refactor:` cambios de código sin nueva funcionalidad
